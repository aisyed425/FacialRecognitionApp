# Steps to follow

Install the opencv-python package with:
pip install opencv-python

Download the haarcascade_frontalface_default.xml pre-trained Haar Cascade model from the OpenCV GitHub repository and store it locally.
Copy the path and replace alg = "haarcascade_frontalface_default.xml" with the appropriate path

Make sure to create a separate folder to have your images placed into.
